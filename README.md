# Poignant

Poignant is a toolkit for natural language processing in Ruby.  It is named in
honor of [why the lucky stiff][1].

## Installation

Add this line to your application's Gemfile:

    gem 'poignant'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install poignant
